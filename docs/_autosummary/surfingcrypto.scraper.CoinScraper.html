surfingcrypto.scraper.CoinScraper

class surfingcrypto.scraper.CoinScraper(coin, fiat, start, end_day, path)

Bases: object

wrapper of GitHub repo cryptocmd.

Scrapes data from coinmarketcap.com.

It scrapes data for the crypto coin specified. Saves data in a *.csv file stored at the provided path. Checks if the file exists, if not it is downloaded. If it exists, it check last element of datetime index and compares it with today’s date. If required, it updates the *.csv file to today’s date.

Parameters

  • coin (str) – symbol of crypto

  • fiat (str) – fiat of prices

  • start (datetime.datetime) – start day

  • end_day (datetime.datetime) – end day

  • path (str) – path to csv file
