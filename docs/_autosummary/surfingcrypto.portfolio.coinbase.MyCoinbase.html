surfingcrypto.portfolio.coinbase.MyCoinbase

class surfingcrypto.portfolio.coinbase.MyCoinbase(active_accounts=True, force=False, *args, **kwargs)

Bases: surfingcrypto.portfolio.coinbase.CB

This class is the user’s Coinbase account.

It stores all cryptocurrency accounts - either active or historic.

It inherits from surfingcrypto.coinbase.CB the client methods and all information regarding the user. It automatically dumps temporary data in order to load accounts faster.

Parameters

  • active_accounts (bool) – default True, select active (balance>0) accounts only.

  • force (bool) – force update from API even if local cache is found.

  • configuration (surfingcrypto.config.config) – package configuration object.
