surfingcrypto.reporting.figures.TaPlot

class surfingcrypto.reporting.figures.TaPlot(trendlines=False, *args, **kwargs)

Bases: surfingcrypto.reporting.figures.BaseFigure

this is the Technical Analysis plot. It shows (at current time): candlesticks, volume and 3 TA Indicators (MACD, BB bands and RSI) Can be easily modified to fit other and/or more indicators.

Parameters

  • trendlines (bool) – UNDER DEVELOPEMENT! - plot also trendlines calculated with src.trend_line class.

  • ts (surfingcrypto.ts.TS) – surfingcrypto.ts.TS object

  • graphstart (str) – date string in d-m-Y format (or relative from today eg. 1 month: 1m,3 month: 3m) from which to start the graph.
