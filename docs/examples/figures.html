Reporting: figures

This module is the core for the production of the images that can be used for visually assessing the price data.

All package figures are based on BaseFigure object that offers basic methods for working with matplotlib figures.

[1]:

%load_ext autoreload
%autoreload 2
from surfingcrypto import Config,TS
from surfingcrypto.reporting.figures import SimplePlot,TaPlot,ATHPlot
